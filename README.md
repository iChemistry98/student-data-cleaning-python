# Student Data Cleaning Script (Python)

This repository contains a simple Python script for cleaning and normalizing
real-world educational survey data stored in CSV format.

## Features
- Standardizes column names
- Removes empty rows and duplicates
- Handles missing values
- Converts numeric-like fields
- Generates a clean dataset and summary report automatically

## Technologies
- Python
- pandas

## How to run
```bash
python clean_students.py
